# BOTDOCS  

## Описание

Проект telegram бота, который хранит заметки и файлы пользователя.  

Функционал следующий:  

- добавление заметок с любым содержимым (текст, фото, видео, аудио, голосовые сообщения), которое поддерживает telegram bot api;  
- удаление заметок;  
- редактирование текстового содержимого заметок;  
- поиск заметок по текстовому содержимому, дате добавления и типу контента.  

### Особенности проекта

1. **Хранение медиа на серверах Telegram.**  Все мультимедийное содержимое (фото, видео, документы и т.д.) будет храниться на серверах платформы Telegram, что позволит существенно съэкономить пространство и ресурсы на девайсе пользователя, а также на сервере проекта. На сервере проекта будут храниться только ID соответствующих файлов.  
2. **Использование SQLAlchemy совместно с aiosqlite.**  Для взаимодействия с базой данных будет использована библиотека SQLAlchemy. Для того, чтобы бот был более отзывчивый, будет использован асинхронный движок aiosqlite для работы с базой данных sqlite.
